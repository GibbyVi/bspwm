Copiar el repositorio:

    git clone https://github.com/GibbyVi/bspwm.git

Copiar la configuracion a las ubicaciones correctas:

    cp bspwm/.config/* ~/
    cp bspwm/.local/* ~/

o mover los archivos:

    mv bspwm/.config/* ~/
    mv bspwm/.local/* ~/


Instalar las fuentes:

    paru -S ttf-raleway
